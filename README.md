## LO-Wallet (Learning Outcome Wallet)

**Status:** (Archived)

*Disclaimer: This repository contains the source code for my university project. It was developed to meet specific academic requirements and is currently no longer actively maintained.*

## About The Project
LO-Wallet is a web-based information system developed for the Academic Resources and Information Technology Center, Phuket Rajabhat University. The system is designed to record and manage the Learning Outcomes of students and university personnel. 

The primary goal is to provide students with a centralized platform to track their training history. Additionally, the e-Certificates generated by the system can be used as supporting documents for their future job applications.

## Key Features
The system supports 3 levels of Role-Based Access Control (RBAC):
*   **User** 
    *   Browse course catalog and view course details
    *   Register for courses
    *   Track course registration history
    *   Download "My Certificates"
    *   Check learning outcomes progression via the "LO-Wallet" menu
*   **Instructor** 
    *   Add and edit course information
    *   View the list of all registered users
    *   View the list of users who successfully completed the course
    *   Manually add late registrations
*   **Officer** 
    *   View the lists of registered users and course completers
    *   Issue e-Certificates to qualified users
    *   Manage digital signatures for certificate stamping
    *   Manually add late registrations

## Tech Stack
*   **Backend:** PHP, Laravel Framework, Livewire
*   **Database:** MySQL
*   **Frontend:** HTML, CSS, JavaScript (Tailwind CSS)

## Database Structure
The system implements a complex Database Schema designed to support specific curriculum requirements, including:
*   Management of Faculties, Majors, and Programs
*   Mapping relationships between Courses and Learning Outcomes
*   Tracking system for Enrollments and Course Completions
